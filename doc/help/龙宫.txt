龙宫的镇宫之宝：perform fengbo 

龙宫练了这么久,感觉风波确实不错,很多人告诉我perform fengbo的
命中率与skills的高低有关,确实如此,经过对源代码的分析,
现在把风波的命中率公布如下: 甲 perform fengbo on 乙,系统作如下计算: 
A=甲的fork_skills等级（注：是fork的有效等级 = fork/2+fengbo-cha） 
B=乙的parry_skills等级（注：同上） 
a=(A*A*A)/10 + random(甲的武学) 
b=(B*B*B)/10 + random(乙的武学) 
判断: if (random(a+b)＞b) {成功} else {失败} 
由此可见,perform fengbo的成功率确实与fork有很大的关系,
但是也需要一定的wx的支持的.而且,这么多的random(),总有可能让你连丫鬟都圈不住,grin 

那成功之后乙的busy时间:  time = A/10,   
if time＜5 then time=5 
if time＞15 then time=15 

也就是说,对方busy的时间与你的fork的等级有关,但最短5秒,最长15秒 

如果不成功的话,自己busy多长呢? 我看的源代码是 time/2,  dao 
如果是这样,自己也死定了,还好,从我用到现在发现,至少我在的站点
自己busy的时间是固定的,那就是对方出三招.我的fork可是 210 哦,所以肯定不是time/2了. 




龙宫的花拳秀腿：perform sheshen 

龙宫轰动一时的舍身技现在已少有人用了，但是他毕竟还是存在，
为了广大的龙宫弟子，我就在此把这一即将绝种的“绝技”进行一次手术，以此纪念！ 

舍身技： 
perform sheshen 

条件： 
dragonfight ＞= 100    dragonforce ＞= 100  有效内力（左边值）＞ 500 

以上条件成立，ok，你可以使用perform sheshen了. 

成功的条件： 

二者有一个成立就可以： 
1、random（对方的有效定力）＜ 15  注：有效定力 = 原始定力 + 内力加力值/2 
2、random（ap + dp）＞ dp 
其中：ap = 自己的unarmed的power   
     dp = 对方的dodge的power/2 

     power的计算方法：  与wx、skills等级、精神最大值、精神当前值有关，（信不信由你） 

     power = （skills_level）^3 
     如果power ＞ 100000 power = power / 精神最大值 * 精神当前值 
     如果power ＜ 100000 power = power * 精神当前值 / 精神最大值 
      这两个有什么区别? 不要问我，我不知道，file里就是这么写的， 
     然后： power = power + wx 

这两个条件成立后，可以向对方发动攻击了，一共三次打击，
但每次打击都会重新判断，以决定该次打击是否成功： 
每一次都作如下运算 
  if （random（ap + pp）＞ pp） 成功 
注意：这里是 pp ： 对方的unarmed的power 

  伤害力计算如下： 
db = 自己的有效臂力 * （100 + 加力值）/ 100   注：有效臂力 = 原始臂力 + 内力加力值 
       伤害值 = 基本伤害值 + （db + random（db））/ 2 
if （random（对方的wx） ＞ 自己的wx） 伤害值 = 伤害值*2/3 

此外，还有wound值，也就是内伤。（谁说perform sheshen不能让对方受伤，且看下文） 
wound = random（伤害值 - 对方防御力） 太惨了，这里还要random，sigh 

其中：基本伤害值第一招100，第二招120，第三招150 

写了这么一大堆，估计能有耐心看的没几个，sigh，简单说吧： 

perform sheshen 的成功率与它们有关： 

1、对方的有效定力：定力值越低，成功率越高，但只影响发起进攻的那一瞬间。 
2、自己的unarmed等级、wx、精神最大值、精神当前值。
这些是用来计算skills_power的，当然是越大成功率越高。而且每一招都要判断。 
3、对方的dodge、unarmed的等级、wx、精神最大值、精神当前值。
当然是越大成功率越低。其中dodge影响总的攻击成功率，unarmed影响每一招。 

伤害力与它们有关： 

1、自己的有效臂力（当然也就与内力加力值有关了） 
2、对方的防御力（当然穿的越多伤害就越小） 
3、random（阿米豆腐） 

最后，perform sheshen的代价：         

很简单：busy = 2 + random（6），也就是 2-7s ，消耗内力300。 

对了，伤害值可以从描述中判断大概：

1、召唤护法的时间间隔： 

time = 60 + (200 - spells的enable等级) 
if time ＜ 30 then time = 30 
所以到了后期，龙出来的越来越快了，不要心疼龙哦，招的越多越好！ 

2、召唤护法的要求： 

法力当前值 ＞ 150，精神当前值 ＞ 80 

3、召唤护法的代价： 

法力当前值 - 150，精神当前值 - 80，busy time 2-4 秒（随机） 

4、召唤护法的成功率： 

random（法力最大值） ＞= 250  成功！

可以保证你不会被困在一个地方（女儿国的绿迷宫除外，是出不来的） 

如果你在fight，系统作如下计算： 
a = 你的spells的enable等级 
b = 对手的spells的enable等级 
还有一个c： 
在水里：c = 50 + random（50） 
在室外：c = 20 + random（20） 
在室内：c = 10 + random（10） 
然后系统判断： 
random（a*c/10）＞ b 成功 
random（a*c/10）＜ b 失败 

3、代价：法力、精神各消耗 50。 

4、目的地：longgong、bibotan的任意一个地方。
